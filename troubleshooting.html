Troubleshooting

Common KatClaw issues and fast fixes. If you’re stuck after trying these steps, email support@katclaw.ai.

Jump to: Install Gateway AI Providers Security Mode Backup & Restore Licensing

Installation Issues

"Git / Xcode Tools install opens Terminal but nothing happens"

Wait 30–60 seconds. macOS may be preparing the installer silently. The install window can appear behind KatClaw Wizard — check the Xcode/Terminal app icon in the Dock and bring that window to front. If no dialog appears, run xcode-select --install manually in Terminal.

"Node.js installation failed"

  1. Download Node.js LTS from nodejs.org
  2. Install the .pkg
  3. Return to KatClaw and click Refresh

Gateway & Web UI Issues

Gateway won’t start

Possible causes: port conflict, broken Node install, or invalid config.

  1. Restart your Mac
  2. Try Start Gateway again
  3. If needed, use Dashboard → Repair Gateway Service

Web UI not loading

  • Use: http://localhost:18789?token=YOUR_TOKEN
  • Get token from Dashboard → VM Network Setup → Step 4, or run openclaw auth token
  • Confirm gateway is running
VM users: direct http://VM_IP:18789 is blocked by browser security. Use SSH tunnel: ssh -L 18789:localhost:18789 user@VM_IP.

AI Provider Issues

API key not working

  • Remove extra spaces
  • Verify key format matches provider
  • Check account credits/subscription

OAuth 2FA stuck on QR/passkey (Claude/OpenAI in VM)

In macOS VMs, Bluetooth passkey flows usually fail.

  1. Click Cancel on QR/passkey
  2. Click Use another method
  3. Choose Authenticator app

Response is empty or weird

Use instruction-tuned models, not reasoning-only models that emit hidden thinking tags.

Security Mode Issues

Commands fail in Moderate mode

Usually you should not need to manually enter binary paths. KatClaw resolves paths automatically, and approval popups can add the exact command/path for you.

Manual full paths are mainly needed for advanced/custom shell commands.

  • Example allowlist entry: /usr/bin/git (binary path only)

Backup & Restore Issues

Backup won’t restore

  • Confirm file is a valid .zip
  • Try a newer backup
  • Make sure KatClaw version is compatible

Settings missing after update

Check auto backups in ~/.openclaw/backups/ and restore Last Known Good.

Licensing Issues

Failed to activate license

  • Check internet connection
  • If active on another Mac, transfer first
  • Contact support@katclaw.ai if still blocked

Founding license page/code not working

Some ISPs block *.workers.dev. Try another network, mobile data, or VPN. Check spam for verification email.
