AI Provider Costs

Understand the ongoing fees for your automation platform.

Important: KatClaw is a one-time purchase, but your agent needs an AI provider to think. This is an ongoing cost — pay-as-you-go API usage based on how much you use your agent.

🔑 Pay-As-You-Go APIs

✨ ChatGPT (OpenAI) Recommended

Subscription / Pay per use

Top default choice for most users. GPT-5.2 is powerful; GPT-5-nano is fast and affordable.

🔀 OpenRouter

Varies by model

One API key, all models — Claude, GPT, Gemini, Llama. Maximum flexibility.

🧠 Claude API (Anthropic)

Pay per use

Best for coding and complex tasks. Direct API access.

🔮 DeepSeek Budget Pick

~$5-20/month

Very affordable. Great reasoning. Works in China.

⚡ Grok (xAI)

Pay per use

Real-time X/Twitter knowledge. 2M token context.

🌙 Kimi (Moonshot)

~$10-30/month

256K context. Affordable. Works in China.

🆓 Free Options

💎 Gemini Free Tier

FREE (with limits)

Google's AI. Generous free tier via AI Studio.

💻 Local LLM Advanced

FREE (runs locally)

LM Studio or Ollama. Needs 32GB+ RAM, M4 Max+.

💡 Our Recommendations

  • Best overall: ChatGPT (OpenAI) — recommended default for most users
  • Most flexible: OpenRouter — one account, all models, maximum flexibility
  • Best for coding: Claude API (Anthropic) — direct access to Claude
  • Best value: DeepSeek API (~$5-20/mo)
  • Free start: Gemini (free tier via API key)
  • China users: DeepSeek or Kimi — no VPN needed

You can switch providers anytime. Start free, upgrade when you need more power. * Pricing as of Feb 2026. Check provider websites for current rates.
