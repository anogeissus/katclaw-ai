Getting Started

What do I need to run KatClaw?

A Mac (Apple Silicon recommended) and an AI provider API key. Some providers like DeepSeek offer very affordable rates — as low as $0.50 per million tokens. See AI provider costs →

How do I install KatClaw?

Download the .dmg file, drag KatClaw to your Applications folder, and launch it. The setup wizard will guide you through the rest.

Does KatClaw work on Intel Macs?

KatClaw requires macOS 14 (Sonoma) or later. It works on Intel Macs, but Apple Silicon is recommended for the best experience with local AI features.

Yes. In KatClaw's menu bar, click Transfer License to Another Mac. This deactivates your current Mac and shows your license key to use on the new one. One Mac at a time.

I already have OpenClaw installed. Why do I need KatClaw?

OpenClaw is powerful but requires command-line setup — editing JSON config files, managing npm packages, configuring security manually, and troubleshooting from the terminal. KatClaw replaces all of that with a native Mac interface:

  • Visual provider management — switch between Claude, GPT, Gemini, DeepSeek and more with one click. No config files to edit.
  • Security controls — three protection levels with a visual settings panel instead of hand-editing exec-approvals.json.
  • One-click updates — update OpenClaw from the menu bar. No terminal commands.
  • Backup & restore — export your entire setup (configs, memories, workspace) to a single file. Restore on any Mac.
  • Auto-start — gateway launches automatically when you log in. No launch agents to configure.
  • Setup wizard — guided installation with prerequisite checks, provider setup, and Telegram configuration in one flow.

Think of it this way: you can manage a web server from the command line, but most people prefer a control panel. KatClaw is that control panel for OpenClaw.

Privacy & Security

Is KatClaw a cloud service?

No. KatClaw is a native macOS application that you download and install on your Mac. There are no cloud servers, no hosted services, and no accounts to create. Everything runs locally on your machine.

Does KatClaw collect my data?

No. KatClaw has no telemetry, no analytics, and no data collection. Your files, configurations, and task logs stay on your Mac. The only network calls KatClaw makes are: (1) license validation, and (2) requests to the AI provider you choose. Full privacy policy →

What are the security controls?

KatClaw provides three security levels for OpenClaw's automation capabilities:

  • Conservative (default): Blocks all system commands. Monitoring only.
  • Moderate: Only allows commands you explicitly approve in an allowlist.
  • Full Access: Unrestricted. For advanced users who understand the risks.

You decide exactly what the automation can and cannot do.

Is this a chatbot?

No. KatClaw is a macOS utility that installs and configures OpenClaw, an open-source automation platform. It manages provider settings, security controls, backups, and gateway operations. It is not a hosted chatbot, conversational AI service, or public-facing bot.
