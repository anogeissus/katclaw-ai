User's Guide

Everything you need to know about working with your AI agent.

Getting Started

Your First Conversation

Open Telegram, find your bot, and type "hi". That's it — your agent will respond.

Your AI agent runs locally on your Mac. When you send a message in Telegram, it travels through a secure gateway to OpenClaw running on your machine. The agent processes your request using the AI provider you configured, and sends the response back to Telegram.

Your Mac needs to be awake and connected to the internet for the agent to respond. KatClaw can auto-start the gateway when you log in, so you don't have to think about it.

Essential Commands

Type these commands in Telegram to control your agent:

/new Start a fresh conversation. Clears the current context so you can begin a new topic. Also works as /reset.
/status Check your agent's status — which AI model is active, token usage, and gateway health.
/model Switch between AI models. Jump from Claude to GPT to Gemini without restarting anything.
/reasoning Toggle extended thinking. When enabled, the agent takes more time to reason through complex problems.

Personalizing Your Agent

Your agent reads a set of markdown files to understand who it is, who you are, and how to behave. These files live in ~/.openclaw/workspace/.

SOUL.md

Defines your agent's personality, tone, and behavior. Want it formal? Casual? Funny? This is where you set that. The agent embodies whatever persona you describe here.

USER.md

Tell the agent about yourself — your name, preferences, work context, timezone. The more it knows, the more helpful it can be without asking.

AGENTS.md

Workspace rules and conventions. How the agent should handle files, memory, safety, and external actions. Think of it as the operating manual.

💡 You can edit these files manually, or just ask your agent: "Update SOUL.md to be more casual and use humor." It can edit its own configuration files.

10 Things to Try

Your agent can do much more than answer questions. Here are some real tasks to try:

  1. "Summarize my unread emails"
    Connects to your email and gives you a quick digest of what's new.
  2. "What's on my calendar today?"
    Checks your calendar and lists upcoming events with times.
  3. "Search the web for latest news on [topic]"
    Searches the web and summarizes what it finds.
  4. "Remind me in 30 minutes to call John"
    Sets a timed reminder that pings you in Telegram.
  5. "Write a Python script that converts CSV to JSON"
    Generates working code and can save it to a file on your Mac.
  6. "What's the weather in Vancouver?"
    Gets the current weather and forecast for any location.
  7. "Read the file at ~/Documents/notes.txt and summarize it"
    Reads files on your Mac and works with their contents.
  8. "Create a new file called todo.md with my task list"
    Creates and writes files directly on your system.
  9. "Check my GitHub PRs"
    Lists your open pull requests, CI status, and review comments.
  10. "Take a screenshot of my desktop"
    Captures your screen and sends the image to Telegram.

Security Modes

KatClaw lets you control exactly what your agent can do on your system. Choose the level that fits your comfort:

🛡️ Conservative Safest

No shell access at all. The agent can chat, search the web, and use built-in tools, but cannot run any commands on your system. Best for getting started.

⚖️ Moderate Balanced

Only allows commands you explicitly approve in an allowlist. The agent can run git status but not rm -rf. You decide exactly what's permitted. Good for daily use.

⚡ Full Access Power Users

Unrestricted system access. The agent can run any command, install packages, modify files anywhere. Only use this if you understand the implications.

💡 You can change security modes anytime in KatClaw's settings. Start with Conservative and move up as you build trust.

Dashboard & Menu Bar

KatClaw lives in your Mac's menu bar. Click the icon to access the dashboard:

  • Start / Stop Gateway — Control whether your agent is online and accepting messages.
  • Agent Status — See which model is active, uptime, and recent activity.
  • Security Audit — Review what commands the agent has run and adjust permissions.
  • Backup & Restore — Export your entire setup (configs, workspace, memories) to a single file. Restore on any Mac with one click.

Troubleshooting

Agent not responding?

Open KatClaw from the menu bar and check if the gateway is running. If it's stopped, click Start Gateway. Also make sure your Mac is awake and connected to the internet.

Commands being blocked?

You're likely in Conservative or Moderate security mode. Check your security settings in KatClaw. In Moderate mode, you can add specific commands to the allowlist.

High token usage?

Long conversations accumulate context. Send /new to start fresh when switching topics. This resets the conversation and reduces token consumption.

API key not working?

Double-check that your API key is valid and has credits. You can update your API key anytime in KatClaw's provider settings without re-running the setup wizard.
